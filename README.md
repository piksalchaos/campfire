# Swimming Over It

Created for Hack Club's [Campfire game jam](https://campfire.hackclub.com/)!

Inspired by the the rage game _Getting Over It_, this game is where you play as a fish who must climb over hills while encountering many obstacles.

## Features:
- Click and hold to propel the fish forward
- Evil anglerfish who will try to obstruct your way
- Created with the [Godot](https://godotengine.org/) game engine
## Ideas we didn't implement in time during the jam
- The fish can't breathe in air, but is also allergic to water, so it can't stay in either for too long
- Narrator that talks to you like in Getting Over It
- the rest of the game lol
